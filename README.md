# 🦋 ArthurSMK - Hub de Jogos & Portfólio

Bem-vindo ao repositório do meu site oficial: **[arthursmk.fun](https://arthursmk.fun)**! 

Este espaço foi criado para centralizar meu portfólio de criação de conteúdo e marketing, além de hospedar os minigames interativos desenvolvidos especialmente para a comunidade das minhas lives.

## 🎮 Jogos Disponíveis

Atualmente, o hub conta com os seguintes projetos:

* **Caça-Erro:** Um quiz interativo onde o objetivo é encontrar a resposta incorreta.
* **O Tabuleiro:** Um board game customizado onde a própria comunidade dita as regras e cria os itens.
* **SMKlicker:** Um jogo clicker baseado nas minhas lives da Twitch.
* *(Mais projetos e interações em desenvolvimento!)*

## ✨ Funcionalidades do Site

* **Modo Claro / Escuro:** Adaptação automática à preferência do sistema do usuário ou via botão (salvando a escolha no `localStorage`).
* **Interface Dinâmica:** Uso de propriedades avançadas como `mask-image` no CSS para adaptar a cor da logo (Branco/Rosa) dependendo do tema ativo.
* **Tags de Status:** Indicadores visuais nos cards para jogos "Novos" e "Em Desenvolvimento".
* **Design Responsivo:** Layout estruturado em CSS Grid para funcionar perfeitamente em desktops, tablets e smartphones.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focado em leveza e performance, utilizando as linguagens base da web:

* **HTML5:** Estrutura semântica dos elementos.
* **CSS3:** Estilização, variáveis de cores (Custom Properties) e animações de hover/clique.
* **JavaScript (Vanilla):** Lógica para gerenciamento do tema visual do site.
